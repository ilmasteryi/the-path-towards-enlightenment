# 定慧等持 (Samadhi-Prajna Balance)

在绝待见地中，定与慧是觉性的一体两面，无需分别修持。

## 定的实相：不随念走

真正的“定”，不是守住一个静止的背景，而是即便在万象森罗中，那份“灵知”从未被境所转。它不是“稳住”了，而是它根本就没动过。

## 慧的实相：知其如是

真正的“慧”，不是掌握了某种深刻的道理，而是不再偏袒任何一种显现。无论升起的是什么，都清清楚楚地知道其“如是”，而不信念头编织的任何剧本。

## 等持：保任的自然状态

当你安住在“知”中，不信念头的内容，也不试图改变当下的状态，这就是定慧等持。

*   **知**：即是慧。
*   **不动**：即是定。

定慧在当下一念中本自等持，无需额外作意。
